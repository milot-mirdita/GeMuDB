Database

The database containing the 28,921,147 × 19 mutations for the human RefSeq sequences is also availabe.

The compressed MySQL Database can be obtained here: GemuDB Database.zip (575 mb).

Import Instructions

The database can be imported as follows: 

unzip "GemuDB Database.zip"
mysql -u<user> -p<password> <database> < "GemuDB Database.sql"

Make sure to replace the entries inside of brackets with your MySQL server information.

/v1/{id}/functionaleffect/detail/{type}/{pos}
getMutationList

GET /v1/{id}/functionaleffect/detail/{type}/{pos}

Get the details of functional effect prediction Returned effects are filtered by position and function effect prediction type.

# URL
http://gemudb.com/api/v1/{id}/functionaleffect/detail/{type}/{pos}
# Parameters

  • path

    Parameter Required Description Data Type
    id true The canonical RefSeq protein accession. string
    pos true The position in the protein that should be returned. int
    type true A comma seperated string containing all functional effect prediction types that should be returned. string
# Response

MutationPosContainer

# Errors
Status Code Reason
404 MutationPos not found.
200 MutationPos
