#API Document ## BasePath: http://gemudb.com/api ## Api Version: v1 ## Resources ### /v1 #### Overview #### `/v1/{id}/functionaleffect/detail/{type}/{pos}` ##### getMutationList **GET** `/v1/{id}/functionaleffect/detail/{type}/{pos}` Get the details of functional effect prediction Returned effects are filtered by position and function effect prediction type. ###### URL http://gemudb.com/api/v1/{id}/functionaleffect/detail/{type}/{pos} ###### Parameters - path
Parameter Required Description Data Type
id true The canonical RefSeq protein accession. string
pos true The position in the protein that should be returned. int
type true A comma seperated string containing all functional effect prediction types that should be returned. string
###### Response [MutationPosContainer](#MutationPosContainer) ###### Errors
Status Code Reason
404 MutationPos not found.
200 MutationPos
- - - #### `/v1/{id}/functionaleffect/detail/{alphabet}/{type}/{pos}` ##### getMutationList **GET** `/v1/{id}/functionaleffect/detail/{alphabet}/{type}/{pos}` Get the details of functional effect prediction Returned effects are filtered by position, alphabet of amino acid one letter codes and function effect prediction type. ###### URL http://gemudb.com/api/v1/{id}/functionaleffect/detail/{alphabet}/{type}/{pos} ###### Parameters - path
Parameter Required Description Data Type
id true The canonical RefSeq protein accession. string
pos true The position in the protein that should be returned. int
type true A comma seperated string containing all functional effect prediction types that should be returned. string
alphabet true A string containing the one letter codes of all aminoacids that should return functional effect predictions. string
###### Response [MutationPosContainer](#MutationPosContainer) ###### Errors
Status Code Reason
400 Parameter Error
404 MutationPos not found.
200 MutationPos

Parameter Required Description Data Type

Parameter Required Description Data Type
q true The query to search by. Query can be a protein accession or amino acid sequence. string
